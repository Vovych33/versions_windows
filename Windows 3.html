Windows 3

Годы поддержки: 1990 - 2008

Что нового *
  • Первый (от Microsoft) удобный для пользователя интерфейс.
  • Появление диспетчера программ.
  • Появление мультимедийных возможностей.
  • Поддержка сети (с 3.1).



    • Имеет значительно обновлённый пользовательский интерфейс, а также технические усовершенствования, позволявшие использовать возможности процессоров Intel 80286 и 80386 по управлению памятью. Текстовые программы, созданные для MS-DOS, могли быть запущены в окне
